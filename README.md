# cosmic-chaos

Your universe awaits! COSMIC CHAOS - an interactive space sandbox where you become a cosmic deity, where you click to spawn stars, double-click to create black holes, drag to paint nebulae, and watch everything interact with gravity physics across 5 trippy visual modes.

<img width="2552" height="1171" alt="image" src="https://github.com/user-attachments/assets/c7c9f897-77a7-4835-8fa9-f2ae3679aa10" />

  Features:
  - Click - Spawn glowing stars that pulse and emit particles
  - Double-click - Create terrifying black holes that devour everything
  - Drag - Paint ethereal nebula clouds across the void
  - Scroll - Zoom in/out of your creation
  - Space bar - Activate TIME WARP (3x speed!)
  - B - Trigger a BIG BANG explosion from the center
  - G - Toggle gravity physics on/off
  - C - Clear the universe and start fresh

  5 Cosmic Modes (keys 1-5):
  1. CREATION - Normal star-forming mode
  2. DESTRUCTION - Chaos reigns
  3. PSYCHEDELIC - Colors shift and flow like a dream
  4. QUANTUM - Particles teleport randomly
  5. VOID - Dark, ominous atmosphere

  Hidden secrets:
  - Stars can go SUPERNOVA when they age (epic flash + explosion)
  - Supernovas have a 30% chance to become black holes
  - Black holes MERGE when they touch
  - Consume 5+ stars with one black hole = secret unlock
  - Konami code (↑↑↓↓←→←→BA) = GOD MODE

  Open it with:
  xdg-open cosmic-chaos.html
